# CodeChef-Inspired Competitive Programming Platform

A full-stack competitive programming platform inspired by CodeChef, designed to help users practice coding problems, improve problem-solving skills, and compete through leaderboards and contests.

## Features

- User Authentication & Authorization
- Coding Problem Management
- Online Code Submission System
- Leaderboard Functionality
- Contest & Challenge Support
- Responsive User Interface
- RESTful API Architecture
- Secure Session Handling

## Tech Stack

### Frontend
- HTML5
- CSS3
- JavaScript
- EJS

### Backend
- Node.js
- Express.js


### Tools
- Git & GitHub
- VS Code

## Project Objective

The goal of this project is to provide an interactive coding environment where users can practice programming problems, participate in coding contests, and track their performance through rankings and leaderboards.

## Key Highlights

- Built scalable backend APIs for coding workflows
- Implemented secure authentication and session management
- Designed leaderboard system for user rankings
- Improved platform usability with responsive UI design
